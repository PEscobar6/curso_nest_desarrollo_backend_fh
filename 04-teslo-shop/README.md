# Teslo API

1. Clonar proyecto
2. Instalar dependencias
```yarn install```
3. Configurar variables de entorno. Renombrar el archivo ```.env.template``` a ```.env```
4. Levantar la base de datos
```
docker-compose up
```
5. Levantar: ```yarn start:dev```
